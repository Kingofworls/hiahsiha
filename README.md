## Usage

**__How To Use Me!?__**

* -> Search for your Movie in [i Popcorn Movies](https://RoyalKrrishna.github.io/iPopcornMovies/) with correct spelling
* -> For YEAR wise movies, type the year. EG: 2022. (You will get all the available movies in 2022)
